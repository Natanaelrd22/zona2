Guía de Configuración Rápida

⚠️ Importante: Esta aplicación necesita configurarse con Supabase para funcionar. Sigue estos pasos antes de usar la aplicación.

1Crear Cuenta en Supabase

  1. Ve a supabase.com
  2. Haz clic en "Start your project" o "Sign In"
  3. Crea una cuenta gratuita con GitHub, GitLab o correo electrónico
Ir a Supabase →

2Crear un Nuevo Proyecto

  1. En el panel de Supabase, haz clic en "New Project"
  2. Configura:
    • Name: Club de Conquistadores
    • Database Password: (elige una contraseña segura)
    • Region: La más cercana a tu ubicación
  3. Haz clic en "Create new project"
  4. Espera 2 minutos a que se configure

3Ejecutar el Script SQL

  1. En el panel izquierdo, haz clic en "SQL Editor"
  2. Haz clic en "New query"
  3. Abre el archivo database/schema.sql de tu proyecto
  4. Copia TODO el contenido
  5. Pégalo en el editor SQL de Supabase
  6. Haz clic en "Run" (o Ctrl+Enter)
✅ ¡Listo! Todas las tablas y políticas fueron creadas

4Crear Usuario Administrador

  1. Ve a Authentication > Users
  2. Haz clic en "Add user" > "Create new user"
  3. Ingresa:
    • Email: ministrylion@gmail.com
    • Password: Admin
    • Email Confirm: ✅ Marca esta casilla
  4. Haz clic en "Create user"
✅ ¡El administrador está creado!

5Obtener Credenciales

  1. Ve a Project Settings (ícono de engranaje ⚙️)
  2. Haz clic en "API"
  3. Copia estos dos valores:
    • Project URL: Se ve así: https://xxxxx.supabase.co
    • anon/public: Una clave larga que empieza con eyJ...

6Configurar el Archivo config.js

Abre el archivo js/config.js y reemplaza:

// Antes (placeholder): const SUPABASE_URL = 'https://placeholder.supabase.co'; const SUPABASE_ANON_KEY = 'placeholder-key'; // Después (tus credenciales): const SUPABASE_URL = 'https://tu-proyecto.supabase.co'; const SUPABASE_ANON_KEY = 'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...';

¡Guarda el archivo!

7¡Ejecutar la Aplicación!

Elige una opción:

  1. Opción 1: Doble clic en index.html
  2. Opción 2: Ejecuta en terminal:
    python -m http.server 8000
    Luego abre: http://localhost:8000
  3. Opción 3: Usa Live Server en VS Code
🎉 ¡Tu aplicación está funcionando!
